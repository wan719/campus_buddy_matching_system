# 校园搭子匹配系统 2.0

## 项目简介
本项目旨在帮助在校大学生找到学习、运动、吃饭等"搭子"，建立真实的校园社交连接。

## 技术栈
- Spring Boot 4.0.4
- Spring Security + JWT
- MyBatis
- MySQL 8.0
- Gradle
