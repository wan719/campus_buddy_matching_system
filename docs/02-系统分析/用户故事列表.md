# 用户故事列表

> 格式：作为 [角色]，我想要 [功能]，以便 [价值]

## P0 级（MVP必须做）

| ID | 用户故事 | 验收标准 |
|----|----------|----------|
| US-01 | 作为学生，我想要用学号注册账号，以便证明我的真实身份 | 学号格式校验、唯一性校验、密码BCrypt加密 |
| US-02 | 作为学生，我想要登录系统，以便访问个人中心 | 支持JWT认证，24小时有效 |
| US-03 | 作为学生，我想要发布搭子活动，以便找到志同道合的人 | 可填类型、时间、地点、人数、描述 |
| US-04 | 作为学生，我想要浏览所有活动，以便选择感兴趣的加入 | 列表展示、分页 |
| US-05 | 作为学生，我想要按条件筛选活动，以便快速找到合适的搭子 | 按类型、时间、地点筛选 |
| US-06 | 作为学生，我想要申请加入活动，以便参与线下见面 | 申请后状态为“待审核” |
| US-07 | 作为活动发起人，我想要审核申请者，以便决定让谁加入 | 通过/拒绝，拒绝时可填理由 |
