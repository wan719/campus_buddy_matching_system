# 用户画像

## 1. 目标用户群体

| 维度 | 描述 |
|------|------|
| **身份** | 在校大学生（本科、研究生） |
| **年龄** | 18-28岁 |
| **场景** | 寻找学习搭子、运动搭子、吃饭搭子、组队参赛 |
| **痛点** | 社交圈有限，找不到志同道合的人；QQ群/频道消息太多，无法精准筛选 |

---

## 2. 典型用户画像

### 用户A：考研党 - 李明

| 属性 | 内容 |
|------|------|
| **基本信息** | 大三学生，男，22岁 |
| **目标** | 找考研搭子一起自习，互相监督 |
| **痛点** | 室友不考研，作息不一致；图书馆一个人容易分心 |
| **使用场景** | 每天晚上发布第二天图书馆占座的需求，找同样考计算机的同学 |
| **期望** | 能按学院、自习地点筛选搭子，有信用分判断是否靠谱 |

---

### 用户B：运动达人 - 王芳

| 属性 | 内容 |
|------|------|
| **基本信息** | 大二学生，女，20岁 |
| **目标** | 找羽毛球搭子，每周固定打球 |
| **痛点** | 身边朋友不喜欢运动，去球场一个人很难组局 |
| **使用场景** | 每周一发布周五下午的羽毛球活动，找3-4个人组队 |
| **期望** | 能按运动类型、场地位置筛选，活动结束后能评价对方 |

---

### 用户C：社团负责人 - 张伟

| 属性 | 内容 |
|------|------|
| **基本信息** | 大四学生，男，23岁，篮球社社长 |

## 3. 用户动机矩阵

| 动机类型 | 占比 | 核心需求 |
|----------|------|----------|
| 学习提升 | 40% | 考研、考公、期末复习搭子 |
| 社交娱乐 | 35% | 吃饭、看电影、游戏搭子 |
| 运动健身 | 20% | 跑步、球类、健身搭子 |
| 组队参赛 | 5% | 竞赛、项目、社团活动 |
