## 1. 设计目标

- 实现高内聚低耦合的分层架构
- 保证系统可扩展性和可维护性
- 采用 RESTful API 设计规范
- 数据库设计满足第三范式

---

## 2. 架构设计

采用经典的三层架构 + 前后端分离：

| 层级 | 技术 | 职责 |
|------|------|------|
| 表现层 | Vue 3 | 页面渲染、用户交互 |
| 控制层 | Spring Boot Controller | 请求接收、参数校验 |
| 业务层 | Spring Boot Service | 业务逻辑处理 |
| 数据层 | MyBatis + MySQL | 数据持久化 |

## 5. 安全设计

| 安全项 | 实现方式 |
|--------|----------|
| 认证 | JWT Token，24小时过期 |
| 授权 | Spring Security 拦截器 |
| 密码存储 | BCrypt 加密 |
| SQL注入 | MyBatis 参数绑定 |
| 跨域 | Spring Boot CORS 配置 |

---

## 6. 部署架构

- 前端：Nginx 托管静态文件
- 后端：Spring Boot 打包为 JAR 运行
- 数据库：MySQL 8.0
- 服务器：云服务器（阿里云/腾讯云）
